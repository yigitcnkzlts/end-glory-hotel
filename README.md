# 🏨 End Glory Hotel - Premium 4-Star Hotel Website

Modern, lüks ve tam özellikli otel web sitesi. Next.js 14 + Spring Boot ile geliştirilmiş, production-ready full-stack proje.

---

## 🌟 Proje Hakkında

End Glory Hotel, Çorlu'da bulunan 4 yıldızlı bir otelin dijital varlığını temsil eden kapsamlı bir web uygulamasıdır. Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, kullanıcı dostu, SEO optimize ve tam responsive bir çözümdür.

### 🎯 Proje Amacı

- Otelin online varlığını güçlendirmek
- 7/24 online rezervasyon imkanı sunmak
- Türkçe ve İngilizce dil desteği ile uluslararası müşterilere ulaşmak
- Modern ve lüks bir kullanıcı deneyimi sağlamak
- Mobil-first yaklaşımla tüm cihazlarda mükemmel performans

- **Flyway** - Migration

- **Swagger/OpenAPI** - API docs

### DevOps
- **Docker** - Containerization
- **Docker Compose** - Multi-container
- **GitHub Actions** - CI/CD
