## 5️⃣ BACKEND API

**Ne Yaptık:**
- Spring Boot 3.x
- PostgreSQL database
- 3 Ana entity:
  1. Room (odalar)
  2. Booking (rezervasyonlar)
  3. ContactMessage (iletişim mesajları)
- RESTful API endpoints:
  - GET /api/rooms (oda listesi)
  - GET /api/rooms/{id} (oda detay)
  - POST /api/bookings (rezervasyon oluştur)
  - POST /api/contact (iletişim mesajı)
- JWT authentication
- Email service (rezervasyon onayı)
- Flyway migration (database versioning)
- Swagger documentation

**Database Schema:**
```sql
- rooms (id, name, type, price, description, amenities)
- bookings (id, room_id, check_in, check_out, guests, status)
- contact_messages (id, name, email, phone, message, created_at)
```
